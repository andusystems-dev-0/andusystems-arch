# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Removed
- Removed youtube-tui and all associated components: Ansible role (`youtube_tui`), config directory (`dotfiles/home/.config/youtube-tui/`), `yt-recommended` script, Hyprland windowrules (`kitty-ws1-youtube`, `yt-recommended`), and the `youtube-tui` + `yt-dlp` packages from `desktop_packages`.
- Removed `mpv` from `desktop_packages` and deleted `dotfiles/home/.config/mpv/`. Waybar's MPRIS module remains functional for other players (e.g. browsers).
- Removed Cursor and Obsidian: dropped leftover `dotfiles/home/.config/{cursor,Cursor,obsidian}/` directories, removed `Cursor`/`cursor` from `dotfiles_exclude`, removed `cursor` + `obsidian` paths from `.gitignore`, and removed the `cursor-bin` row from `docs/packages.md`.
- Removed all screenshot utilities: deleted `dotfiles/home/.config/flameshot/`, `dotfiles/home/.local/share/applications/org.flameshot.Flameshot.desktop`, `dotfiles/home/.local/bin/screenshot.sh`, all `Print` keybinds from `hyprland.conf`, the `screenshot.sh` chmod task from the theming role, and the `flameshot` entry from `docs/packages.md`. (`flameshot`, `grim`, `slurp` were never in Ansible package lists.)

## [2026-04-10]

### Changed
- Enhanced `.gitignore` and Ansible dotfiles exclude list to cover additional sensitive directories — IDEs (Cursor, Code, VSCodium, JetBrains), cloud CLI credentials (helm, kubectl, gcloud, azure, argocd, doctl), browser profiles, chat clients, shell history, and generic secret patterns (*.pem, *.key, .env).
- Aligned `.gitignore` exclusions with `dotfiles_exclude` in Ansible defaults for defense-in-depth.

### Removed
- Removed duplicate `.gitignore` entries.
- Removed obsolete standalone color configuration files for Hypr, Kitty, Nvim, Rofi, and Waybar (now generated by matugen at runtime).
- Removed control-panel, clock-widget, and weather-widget scripts and their associated styles.
- Cleaned up associated Ansible theming task references for removed scripts.
- Removed stats panel from desktop layout.

## [2026-04-01]

### Changed
- General updates to configuration files.

## [2026-03-31]

### Added
- youtube-tui integration with yt-dlp cookie-based authentication via Zen Browser profile detection.
- youtube-tui Ansible role with automated yt-dlp config generation (cookie auth with fallback).
- Waybar status bar configuration with MPRIS media player integration and clock module.
- Neofetch system info display configuration.
- Workspace repository cloning role — clones repos from internal git host with GitHub mirror remotes.
- Desktop packages role with additional fonts (Hack, Cascadia Code, FiraCode, JetBrains Mono Nerd Fonts, Noto, Inter) and Papirus icon theme.
- Waybar menu and panel configuration.

### Changed
- Refactored workspace layout — Neovim opens full-screen on workspace 2, youtube-tui on workspace 1.
- Removed stats panel and replaced with cleaner Waybar-only layout.
- Updated `.gitignore` to exclude Python bytecode, Ansible-generated configs, and wallpapers.
- Fixed Zen Browser to render at 100% opacity via Hyprland window rule.

## [2026-03-30]

### Added
- Matugen-based Material You theming pipeline — generates colors for Hyprland, Kitty, Waybar, Rofi, Neovim, GTK 3/4, and swaync from wallpaper images.
- Hyprland theming integration with dynamic color variables sourced from matugen templates.
- Neovim color integration via mini.base16 with matugen-generated palette.
- Rofi application launcher with Wayland-native click-to-close behavior.
- App cleanup Ansible role — removes htop/dolphin/vim, installs btop, hides unwanted launcher entries.
- Lazygit integration with LazyVim.
- PipeWire audio stack with PulseAudio and ALSA compatibility layers.
- Screenshot support via Flameshot.
- Sound functionality and media player controls.

### Changed
- Changed wallpaper directory from `wallpapers/` to `Wallpapers/`.
- Switched to Neovim as sole editor (removed vim dependency).

### Fixed
- Fixed Rofi launcher closing behavior on Wayland (fullscreen transparent layer surface).
- Fixed control panel Neovim color reload command.

## [2026-03-30] - Initial Release

### Added
- Initial Arch Linux Ansible playbook structure with localhost-only execution.
- Core packages role with yay AUR helper build from source.
- Dotfiles management via GNU Stow with automatic adoption of unmanaged configs.
- Hyprland Wayland compositor configuration with multi-monitor support.
- Kitty terminal emulator configuration.
- Rofi application launcher.
- Bootstrap script for fresh Arch Linux installs (`scripts/bootstrap.sh`).
