# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

## [2026-04-10]

### Changed
- Enhanced `.gitignore` and Ansible dotfiles exclude list to cover additional sensitive directories — IDEs, cloud CLI credentials, browser profiles, chat clients, shell history, and generic secret patterns.
- Aligned `.gitignore` exclusions with `dotfiles_exclude` in Ansible defaults for defense-in-depth.

### Removed
- Removed duplicate `.gitignore` entries.
- Removed obsolete standalone color configuration files for Hypr, Kitty, Nvim, Rofi, and Waybar (now generated by matugen at runtime).
- Removed control-panel, clock-widget, and weather-widget scripts and styles.
- Cleaned up associated Ansible theming task references.

## [2026-04-01]

### Changed
- General updates to configuration files.

## [2026-03-31]

### Added
- youtube-tui integration with yt-dlp cookie-based authentication via Zen Browser.
- youtube-tui Ansible role with automated yt-dlp config generation.
- Waybar status bar configuration and menu.
- Neofetch system info display.
- Workspace repository cloning playbook (internal Git host with GitHub mirror remotes).
- Desktop packages role with additional fonts (Nerd Fonts, Noto, Inter) and Papirus icon theme.

### Changed
- Refactored workspace layout — Neovim opens on workspace 2, youtube-tui on workspace 1.
- Removed stats panel and replaced with cleaner Waybar layout.
- Updated `.gitignore` to exclude Python bytecode, Ansible-generated configs, and wallpapers.
- Fixed Zen Browser to render at 100% opacity.

## [2026-03-30]

### Added
- Matugen-based Material You theming pipeline — generates colors for Hyprland, Kitty, Waybar, Rofi, Neovim, GTK 3/4, and swaync from wallpaper images.
- Hyprland theming integration with dynamic color variables sourced from matugen.
- Neovim color integration via mini.base16 with matugen-generated palette.
- Rofi application launcher with click-to-close behavior on Wayland.
- App cleanup Ansible role — removes htop/dolphin/vim, installs btop, hides unwanted launcher entries.
- Lazygit integration with LazyVim.
- PipeWire audio stack with PulseAudio and ALSA compatibility layers.
- Screenshot support via Flameshot.

### Changed
- Changed wallpaper directory from `wallpapers/` to `Wallpapers/`.
- Switched to Neovim as sole editor (removed vim dependency).

### Fixed
- Fixed Rofi launcher closing behavior on Wayland (fullscreen transparent layer surface).
- Fixed control panel Neovim color reload command.

## [2026-03-30] - Initial Release

### Added
- Initial Arch Linux Ansible playbook structure.
- Core packages role with yay AUR helper build.
- Dotfiles management via GNU Stow.
- Hyprland Wayland compositor configuration.
- Kitty terminal emulator configuration.
- Rofi application launcher.
- Bootstrap script for fresh Arch Linux installs.
