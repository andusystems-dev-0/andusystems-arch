# Wallpapers

Place your wallpaper images in this directory (`~/Wallpapers/`). Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`.

The `theme-switch.sh` script picks a random image from here, sets it via `swww`, and runs `matugen` to generate a matching color scheme for all apps.

Image files are gitignored — only this README and `.gitignore` are tracked.
